# Rump

Rump is a designed for the study of L-algebras. It provides tools for constructing L-algebras, checking their properties (e.g., primality, ideals, subalgebras), and performing algebraic operations such as direct products and semidirect products.

Created with Pkg

To use it
1. start Julia from the package directory (.Rump)
2. go to the package mode:
```julia-repl
    julia> ]
    (Rump) pkg>
```
3. activate the package:
```julia-repl
    (Rump) pkg> activate .
```
4.  exit the pkg mode with backspace

5. use the package:
```julia-repl
    julia> using Rump
```
